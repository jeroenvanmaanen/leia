Provides a sub-command that consolidates {@link org.leialearns.api.enumerations.ModelType#TOGGLED} versions into a new {@link org.leialearns.api.model.Expected} object. An Expected object is a model of the statistically significant aspects of the behavior of the environment given the current history of the interaction between LEIA and its environment.

The the most recent Expected object forms the basis for the Maximizer to update the {@link org.leialearns.api.enumerations.ModelType#POLICY} that is used by the {@link org.leialearns.command.encounter encounter} to interact with the environment.
