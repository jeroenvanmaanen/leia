Provides a sub-command that consolidates {@link org.leialearns.api.enumerations.ModelType#COUNTED} versions into a new {@link org.leialearns.api.model.Observed} object. An Observed object has both a {@link org.leialearns.api.enumerations.ModelType#OBSERVED} and a {@link org.leialearns.api.enumerations.ModelType#DELTA} version associated with it. An Observed object is a model of the behavior of the environment given the current history of the interaction between LEIA and its environment.

The most recent Observed object forms the basis for the {@link org.leialearns.command.minimizer minimizer} to update the model of expected behavior.
