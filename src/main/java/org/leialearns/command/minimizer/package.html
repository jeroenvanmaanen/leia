Provides a sub-command that evaluates whether nodes should be added to, or removed from, the model of expected behavior, in order to minimize the total description length of the interaction so far. If a node should be added or removed, a new {@link org.leialearns.enumerations.ModelType#TOGGLED} version is created for that operation.

Toggled versions are consolidated by the {@link org.leialearns.command.consolidator consolidator} into a new {@link org.leialearns.logic.model.Expected} model version.
